Latest Builds ============= These are the very latest builds built by [Travis](https://travis-ci.org/Jadar/RankCapes). There is a build every time that a change is pushed to [GitHub]({{ site.github-project }}). For this reason, **even though a build might have succeeded, there most likely will still be bugs in it.** I try not to push bad code but it happens every once and a while.
